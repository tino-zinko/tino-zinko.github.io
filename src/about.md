# 2. ランキング機能

現在、滞在時間のみからランキングを算出しています。

訪問回数、コメント数、ハート数、スプーン数などはランキングに一切影響を与えません。

また、ユーザー名を変更してもランキングの情報は保持されます。

# 3. コマンド

コマンド一覧は`/help`で確認できますが、一部、隠しコマンドがあります。

## 3.1 声のエフェクトに関するコマンド

- `/echo`: エコーをかける

- `/asmr`: こそこそ声にする

- `/high`: 声を高くする

- `/low`: 声を低くする

- `/fast`: 早口にする

- `/slow`: 遅口にする

## 3.2 声の種類に関するコマンド

- `/zundamon`

- `/zundamon_2`

- `/zundamon_3`

- `/zundamon_4`

- `/zundamon_5`

- `/asmr`

- `/sayo`

- `/tsumugi`

- `/himari`

- `/nurse`

- `/nurse_asmr`

- `/bii`

- `/bii_calm`

- `/bii_shy`

## 3.3 それ以外のコマンド

- `/bgm`: 枠のBGMを一時的に変更する

# 4. よくある質問

Q. 回答が`ERROR`となる条件は何ですか

A. AIの内部でエラーが発生したときに`ERROR`と表示されます。時間をおいて同じ質問をすると正しく回答が得られる場合があります。
